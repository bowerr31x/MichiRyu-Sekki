# MichiRyu Content Libraries

This directory documents the intended private content-root layout for the hosted MichiRyu Content API.

Do not commit proprietary content here. Stories, artwork, maps, icons, PDFs, educational materials, and Yuki no Sato content should be deployed to the private server-side content roots only.

Recommended hosted layout:

```text
michiryu-content-libraries/
  basic/
  custom/
  premium/
```

Each library root should contain its own `featured-content.json`, `images.json`, and referenced asset folders.
