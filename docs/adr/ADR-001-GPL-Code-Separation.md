# ADR-001

Title:
Separate MichiRyu software from proprietary content

Status:
Accepted

Date:
2026-06-15

Decision:

MichiRyu-Sekki shall contain GPL software only.

Stories, artwork, maps, educational content,
and character content shall be stored separately
and accessed through a content provider architecture.

Rationale:

- Protect intellectual property
- Allow WordPress.org distribution
- Enable future premium content
- Simplify licensing
